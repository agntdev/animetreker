# АнимеТрекер

Персональный Telegram-бот для одного пользователя: быстрые кнопки для отметки эпизодов, история просмотров, прогресс по сезонам и опциональные оповещения о новых сериях.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
